Sort YouTube Playlist by Duration

Sort any playlist you own by video length, shortest or longest first, in seconds. No extension to install.

1. Drag this button to your bookmarks bar:

▶ Sort Playlist

Tip: if your bookmarks bar is hidden, press Ctrl+Shift+B (Windows) or ⌘+Shift+B (Mac) to show it.

2. Use it on YouTube:

  1. Open a playlist you own (or your Watch Later).
  2. Set the playlist's “Sort by” to Manual. YouTube only allows reordering in Manual mode.
  3. Click the ▶ Sort Playlist bookmark. The sorting panel appears below the playlist header.
  4. Pick shortest- or longest-first and hit Sort.

It always loads the latest version straight from GitHub, so you never have to update anything.

Prefer to add it manually? Copy the bookmarklet code

Create a new bookmark, name it “Sort Playlist”, and paste this as the URL.
